# AGENTS.md

## 敵弾

* 敵弾はこの `Bullets` 配下に作成してください。
* クラス名は `Bullet_(固有名)` としてください。
* 敵弾のクラスは、必ず `BulletBase` を継承してください。
* コンストラクタでは、必ず `this.Initialize_Bullet` を呼び出してください。
* ラーニング可能な敵弾は `Initialize_Bullet` の第三引数を `true` にしてください。それ以外の敵弾は第三引数を `false` にしてください。
* ラーニング可能な敵弾を追加する場合は、`Game/HLTGame/HLTGame/Games/Gameplays/WeaponLarning.cs` の `Attack` メソッドと `Annihilate` メソッドにも登録してください。登録方法は各メソッドの既存実装を参考にしてください。
* 敵弾の画像は `Pictures.Dummy` を使用してください。画像は後でユーザーが差し替えます。
* 敵弾は、プレイヤーが赤い弾を避け、青い弾を吸収し、ラーニングした同種攻撃で対消滅・反撃するという `spec/annihilation企画書.txt` のコンセプトに沿って設計してください。
* 新たに敵弾を追加する場合は、この名前空間配下にある `Ordinary` と名の付く既存クラスを、書き方の参考にしてください。これらの参考クラスは今後増える可能性があります。
* 既存の敵弾実装、命名規則、配置規則に合わせて、局所的に実装してください。
