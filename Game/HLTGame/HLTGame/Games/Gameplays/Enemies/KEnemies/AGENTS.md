# AGENTS.md

## 敵キャラクタ

* 敵キャラクタはこの `KEnemies` 配下に作成してください。
* クラス名は `KEnemy_(固有名)` としてください。
* 敵キャラクタのクラスは、必ず `KEnemyBase` を継承してください。
* 敵キャラクタの画像は `Pictures.Dummy` を使用してください。画像は後でユーザーが差し替えます。
* 新たに敵キャラクタを追加する場合は、この名前空間配下にある `Ordinary` と名の付く既存クラスを、書き方の参考にしてください。これらの参考クラスは今後増える可能性があります。
* 既存の敵キャラクタ実装、命名規則、配置規則に合わせて、局所的に実装してください。
