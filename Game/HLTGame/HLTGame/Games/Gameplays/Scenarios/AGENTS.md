# AGENTS.md

## ステージシナリオ

* ステージシナリオはこの `Scenarios` 配下に配置してください。
* ステージ 1～6 は、それぞれ `StageScenario_NN` というクラス名にしてください。
* `E_EachFrame` で敵を追加してください。
* `E_EachFrame` の列挙戻り値は `int` です。
* `E_EachFrame` で返したフレーム数のぶん待機します。
* `0` を返すとステージ終了です。
* `E_EachFrame` メソッドが終了することで、そのステージの終了を表します。
* 新たにステージシナリオを追加する場合は、この名前空間配下にある `Ordinary` と名の付く既存クラスを、書き方の参考にしてください。これらの参考クラスは今後増える可能性があります。
* 既存のステージシナリオ実装、命名規則、配置規則に合わせて、局所的に実装してください。
