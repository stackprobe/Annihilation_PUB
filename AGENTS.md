# AGENTS.md

## 修正対象

* 原則として、修正を行うのは `Game/HLTGame/HLTGame/` と `story/` のみです。
* それ以外のフォルダは、ユーザーから明示的な指示がある場合だけ変更してください。
* フォルダ構成は変わる可能性があるため、不要なフォルダ説明や構成の前提を追加しないでください。
* サブディレクトリに別の `AGENTS.md` がある場合は、対象ファイルに近い方の指示を優先してください。

## 実装方針

* 既存の C# / WinForms プロジェクト構成と命名規則に合わせて実装してください。
* ゲーム本体の実装は主に `Game/HLTGame/HLTGame/` 配下を確認してください。
* ストーリー関連の修正は `story/` 配下を確認してください。
* 仕様や挙動が不明な場合は、既存コードとユーザーの指示を優先して判断してください。
* 指示がない大規模リファクタリング、無関係な自動整形、一括整形は避けてください。

## ビルド

* ゲーム本体のテストビルドは `Game/HLTGame` をカレントディレクトリにして `TestBuild.bat` を実行します。
* `Game/HLTGame/TestBuild.bat` を使う前に、`dotnet build` や任意の `MSBuild` コマンドを直接実行しないでください。
* `Game/HLTGame/TestBuild.bat` はユーザーから明示的な指示がない限り変更しないでください。
* ルートの `ReleaseAll.bat` はリリース成果物の作成、暗号化、コピー、アップロードまで行うため、ユーザーから明示的な指示がない限り実行しないでください。
* `Release.bat` と `Clean.bat` は実行禁止です。

## 文字コードと改行

* 既存ファイルの文字コードと改行コードは維持してください。
* `.cs` ファイルは UTF-8 with BOM / CRLF とします。
* `.txt` ファイルは Shift_JIS (CP932) / CRLF とします。
* `.bat` ファイルは Shift_JIS (CP932) / CRLF とします。
* `.ps1` ファイルは UTF-8 / CRLF とします。
* 無関係な文字コード変換や改行コード変換は避けてください。
