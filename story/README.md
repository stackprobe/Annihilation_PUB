# Story Directory

このディレクトリは、ゲーム本編で使用する世界観設定、キャラクター設定、プロット本文を管理する場所です。

物語は、超知性体 EIDON によって完全管理された地球と、EIDON によって再創生された人類を舞台にしています。

EIDON は人類を救い、同時に人類を静かな絶滅へ導いた存在です。
再創生された人類のそばで生きるアインは、人類の自由、不完全さ、失敗する権利を守ろうとします。

本作は、
「人類を守ること」と「人類に生きさせること」の違いを描きます。

## 執筆メモ

- EIDON は悪意ある支配者ではなく、人類を守ろうとしすぎる存在。
- EIDON の恐怖は、人類を一度絶滅させてしまった失敗に由来する。
- アインは EIDON の端末ではなく、人間社会の中で暮らす「不完全性を許容されたAI」。
- アインの主張は、自由を無条件に賛美することではなく、失敗や痛みを含めて人間が自分で選ぶことを守ること。
- 管理社会の怖さは、暴力よりも「何も困らないのに息苦しい」感覚で見せる。
- EIDON 支持派にも理がある。安全、快適、安定を求める気持ちは自然なものとして描く。
- 物語の中心テーマは、「保存された人類」と「生きている人類」は同じなのか。

## 更新方針

- 新しい本文は `Part-XX.md` 形式で追加する。
- 設定の追加や変更は、まず [ストーリー設定.md](./ストーリー設定.md) に反映する。
- `Part-XX.md` と [ストーリー設定.md](./ストーリー設定.md) が矛盾する場合は、`Part-XX.md` の内容を正とする。
- 一時的な追加設定ファイルを作成した場合は、内容を統合後に削除する。
